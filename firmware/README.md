# Firmware

Servo control, gait logic, sensor integration, safety behavior, telemetry, and embedded code.

## Early priorities

1. Servo calibration utilities.
2. Static pose control.
3. Stand/sit transitions.
4. Basic walking gait.
5. Emergency stop behavior.
6. Telemetry reporting.

## Rule

No advanced personality, voice, or fleet behavior until Alpha can stand and move reliably.
