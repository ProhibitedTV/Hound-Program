# Mechanical

CAD, OpenSCAD, printable parts, revision notes, and print instructions.

## Current prototype areas

- `heads/alpha/` — Hound Alpha alien-animal head shells and expression armor.
- `feet/alien-hoof-type-a/` — modular consumable foot core and Flex Seal coating experiments.

## Mechanical rules

- PLA+/Tough PLA is the default print material for the Kobra 2 Max.
- PETG is not assumed reliable on the current machine.
- Printed parts should locate, house, protect, or express; carbon fiber/metal should carry difficult load paths when needed.
- Every part should eventually get a revision ID and print notes.
