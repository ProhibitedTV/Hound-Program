# Alpha v0.3 Quick Notes

Open `hound_alpha_v0_3_assembly.scad` and start with:

`PART = "leg_layout"`

Then try:

`PART = "assembly"`

v0.3 focuses on fixing the leg read:

- hip pod
- upper link
- knee
- lower link
- ankle/pastern
- hoof

The ankle/pastern can be fixed or passive for Alpha. It exists because the hoof should not look like it is glued directly to a vertical stick.
