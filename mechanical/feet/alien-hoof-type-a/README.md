# Alien Hoof Type A

OpenSCAD printable modular foot system for the Hound Program.

## Design intent

- PLA/PLA+ printable consumable foot core.
- Flex Seal / rubberized coating friendly geometry.
- Tri-lobed alien hoof contact patch.
- Universal 2x M3 mounting interface.
- Batch-manufacturable replacement feet.

## First test

1. Open `hound_alien_hoof.scad`.
2. Set `PART = "foot_core";`.
3. Render/export STL.
4. Print in PLA+/Tough PLA.
5. Test bare first, then Flex Seal coating workflow.

## Initial suggested print settings

- Material: PLA+ / Tough PLA
- Layer height: 0.2 mm
- Walls: 5+
- Infill: 35–50% gyroid/cubic
- Supports: avoid if possible

## Notes to collect

- Weight.
- Print time.
- Grip on tile, concrete, carpet.
- Sound on hard floors.
- Flex Seal adhesion and wear.
