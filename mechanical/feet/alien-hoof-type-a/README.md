# Alien Hoof Type A

OpenSCAD printable modular foot system for the Hound Program.

## Files

- `hound_alien_hoof.scad` — original v0.1 concept.
- `hound_alien_hoof_v02.scad` — manufacturing-minded v0.2 revision.

## Design intent

- PLA/PLA+ printable consumable foot core.
- Rubberized coating friendly geometry.
- Tri-lobed alien hoof contact patch.
- Universal 2x M3 mounting interface with 28 mm spacing.
- Batch-manufacturable replacement feet.

## Recommended first test

1. Open `hound_alien_hoof_v02.scad`.
2. Set `PART = "test_coupon"`.
3. Print the coupon and test coating adhesion.
4. Set `PART = "foot_core"`.
5. Print one bare foot core.
6. Test bare grip first, then coating workflow.

## v0.2 improvements

- Stronger top mounting bosses.
- Clearer 28 mm M3 interface.
- Better coating retention rim.
- Dedicated batch-four preview mode.
- Dedicated adapter mock.
- Cleaner tread cuts for early grip testing.

## Initial suggested print settings

- Material: PLA+ / Tough PLA
- Layer height: 0.2 mm
- Walls: 5+
- Infill: 35–50% gyroid/cubic
- Supports: avoid if possible

## Notes to collect

- Weight.
- Print time.
- Grip on tile, concrete, carpet.
- Sound on hard floors.
- Coating adhesion and wear.
- Whether the mounting bosses need more diameter or height.
