# Alpha Print Readiness

Alpha is still in concept CAD. The current goal is to make the render worth turning into physical test parts.

Before any large print, create small validation pieces first.

## Minimum first prints

1. Fastener gauge.
2. Insert socket gauge.
3. Hoof coating coupon.
4. Hoof core.
5. Ankle and hoof interface mock.
6. One leg module mock.
7. One body panel sample.

## Required part notes

Every print candidate needs:

- material
- orientation
- wall count
- infill target
- support expectation
- screw list
- insert list
- mating part list
- service access notes

## Design constraints

- PLA+ or Tough PLA baseline.
- Avoid unsupported flat overhangs.
- Split large shells.
- Keep screws reachable.
- Keep electronics accessible.
- Keep decorative armor separate from structure.
- Validate tolerances with coupons before relying on them.
